== Description ==
Worldpay US links merchants with their Worldpay US accounts, allowing them to take payments on their wordpress sites while remaining SAQ A compliant.
This plugin is the official Worldpay US plugin and has been certified by Worldpay US. Worldpay is one of the leading payment processing companies in the industry today. 
Increase your revenue and customer satisfaction by offering secure credit card and ACH payments! If you don't already have a Worldpay US account, 
simply go to <a target="_blank" href="https://www.worldpay.us/partner/Bradstreet-Payment-Plugins">Worldpay US</a> and fill out the merchant signup form. With the convenience of credit card and ACH,
all your payment processing needs can be satisfied with just one plugin.


We encourage all potential customers to sign up for a <a href="http://www.securenet.com/developers">Worldpay US Sandbox Account</a>
so they can test the plugin's functionality. If you have any questions or concerns please contact support@paymentplugins.com for assistance.

If you want to sell subscription products but don't have the WooCommerce Subscriptions plugin, you can use the built in subscription plugin functionality.

<h3>Requirements</h3>
In order to use this plugin, you must have a <a target="_blank" href="https://www.worldpay.us/partner/Bradstreet-Payment-Plugins">Worldpay US</a> account and a <a target="_blank" href="https://paymentplugins.com/signup">Payment Plugins</a> account. Once you sign up for a Worldpay US account, you can enter your API
keys in the Payment Plugins site and begin testing the plugin. For detailed instructions, please see the tutorials page of the plugin once you install it. 

= Features =
- SAQ A PCI Compliant
- Accept Credit Card Payments
- Accept ACH/Bank Payments
- Refund Orders
- WooCommerce Integration
- WooCommerce Subscription Integration
- Subscription functionality built in to plugin


<h3>Installation Instructions</h3>
<p>This is a quick reference guide that explains how to setup the plugin.</p>
<ul>
  <li>Sign up for a <a target="_blank" href="https://securenet.secure.force.com/UMA?partnerCode=9280">Worldpay US account</a>.</li>
  <li>Sign up for a <a target="_blank" href="https://paymentplugins.com/signup">Payment Plugins</a> account.</li>
  <li>Once Worldpay approves your merchant account, login to your Worldpay terminal and get your Securenet ID, Secure Key, and Worldpay Public Key.</li>
  <li>Login to <a target="_blank" href="https://paymentplugins.com/login">Payment Plugins</a> and navigate to the API keys page. Enter your Securenet ID, Secure key, and Worldpay Public Key.</a></li>
</ul>

Login to your <a href="https://terminal.securenet.com/login.aspx">Worldpay terminal</a> and navigate to the settings page. From there, you can generate your API keys.
